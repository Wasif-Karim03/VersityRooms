# Public Assets

## Slocum Hall Image

Please add your Slocum Hall image file here as:
- `slocum-hall.jpg` (recommended format: JPG, optimized for web)

The image will be used as the hero background on the landing page.

### Image Requirements:
- Format: JPG or PNG
- Recommended size: 1920x1080 or larger
- File size: Optimized for web (under 500KB recommended)

### How to Add:
1. Save your Slocum Hall image
2. Place it in this `public` folder
3. Name it exactly: `slocum-hall.jpg`

The application will automatically use this image as the hero background.
